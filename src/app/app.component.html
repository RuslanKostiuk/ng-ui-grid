Angular standalone grid

Sortable headers, header filters, column menus, context menus, and infinite scroll.

The same grid component runs in local mode and server mode. Local mode sorts and filters in-memory. Server mode pushes sort, filter, and paging state into a callback that returns the next slice.

Local data with infinite scroll

Client-side sorting and filtering over 600 generated rows.

Server-side paging and filtering

Infinite scroll requests pages from a simulated backend using the current sort and filter state.
